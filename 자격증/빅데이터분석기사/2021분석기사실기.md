4. 기타

- 시험은 오픈북이 아니며, 별도의 레퍼런스 제공없이 응시환경 내에서만 작업 가능
  (단, 응시환경 내 help 함수 이용 가능)

- 답안 제출방법
  - 단답형 : 정답 입력 후 제출
  - 작업형 제1유형 : 단답형 답을 가진 변수를 print 명령어로 출력하는 코드 제출
  - 작업형 제2유형 : write.csv, to_csv 명령어를 이용하여 답안을 csv로 생성하는 코드 제출
